# ReactJS + Hooks + Redux + Json Server - Json Server Auth - Ant Design - Building an app ecommerce De Vinc website.

## Requirements

* **nodejs >= v18.12.1**
* **npm >= 8.19.2**
* **yarn >= 1.22.19**

Clone project and run test.

```
$ git clone https://git_url_clone <project_dir>
$ cd <project_dir>
$ yarn install
$ yarn start
$ yarn run server
```
